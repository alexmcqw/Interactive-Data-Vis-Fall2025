# Lab 3: Mayoral Mystery

This dashboard analyzes a political campaign's performance in the 2024 NYC mayoral election.

## Dashboard Features

The dashboard includes four visualizations:

1. **Election Results Map**: Choropleth map showing candidate vote percentage by district
2. **Income Category Analysis**: Bar chart comparing performance across income categories
3. **GOTV Effectiveness**: Scatter plot examining the relationship between door-knocking efforts and election outcomes
4. **Policy Alignment**: Grouped bar chart showing policy alignment scores by voting behavior

## Files

- `index.md`: Observable-style markdown notebook with all visualizations
- `index.html`: HTML version for GitHub Pages deployment
- `data/`: Contains all data files (nyc.json, election_results.csv, survey_responses.csv, campaign_events.csv)
